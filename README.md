# MailJS
Full blown mail system written in NodeJS.

---
### Features:
* SMTP server
* RESTfull api.
* Oauth 2.0
* Scalable
* Intergrated web app
* Compleatly secured with HTTPS/TLS
* Automatical certificate installation with 
* Multiple domains
